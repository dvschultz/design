    Digital Books

    All books used the Inkling platform, which is a system of xml and html files used to generate a book. I designed and developed the first title from start to finish with the author and ceo/editor. Subsequent titles were designed with another developer and sometimes an intern. I collaborated with the cto on a php script that would convert Google Docs (where all of our stories began) into an Inkling-based title including file structure and interactive components using a markdown-like syntax for headers and interactive elements. I also created a component and style guide with junior developer and a series of sass mixins to customize the look of each book while built on the same html structure. Additionally I designed numerous marketing sites for the company.
