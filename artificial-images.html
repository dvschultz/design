    Teaching Machine Learning to Image Makers

    In the past year I have been teaching a class titled Artificial Images. This class is taught to working designers and image makers to give them a taste of generative machine learning models. So far the class has been taught twice at SSHH, with a third upcoming and a fourth class for remote participants in the works.

    The class covers some high level machine learning theory, but is mostly focused on learning through making. During the 4-week class students learn style transfer, CycleGAN, next frame prediction (through Pix2Pix), and StyleGAN. These models are taught using a mix of Google Colab, Runway, and cloud GPU servers.

    All of the materials from these classes are recorded and made available to anyone on YouTube. If you’re interested in learning more about my classes, you can sign up for notifications of new classes and new videos here.
